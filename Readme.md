# Assignment 2 – Banking System

Student: Darkhan Daurambekov | SE-2512

---

## About the project

In this assignment I created a simple banking system using Java.  
The goal was to understand how different data structures work in real situations.

I used LinkedList, Stack, Queue and Array.

---

## Task 1 – LinkedList (Accounts)

I created a BankAccount class and used LinkedList to store accounts.  
Program can:
- add account  
- display accounts  
- search account  

---

## Task 2 – Deposit & Withdraw

I added:
- deposit money  
- withdraw money  

Balance updates directly in the list.

---

## Task 3 – Stack (History)

I used Stack to store transactions.

- push → add  
- pop → undo  
- peek → last transaction  

---

## Task 4 – Queue (Bills)

Queue is used for bill payments.

- add bill  
- process bill  
- display queue  

FIFO principle.

---

## Task 5 – Queue (Requests)

User sends request instead of creating account directly.

- request goes to queue  
- admin processes it  
- account added to main list  

---

## Task 6 – Array

Created array with 3 accounts.

Fixed size, unlike LinkedList.

---

## Menu

Program has:

Bank:
- submit request  
- deposit / withdraw  
- view accounts  

ATM:
- check balance  
- withdraw  

Admin:
- process requests  
- process bills  

---

## What I learned

- how LinkedList works  
- difference between Stack and Queue  
- how to apply data structures  
- how to build simple Java system
